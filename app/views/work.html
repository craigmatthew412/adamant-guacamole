Mayberry Hospitality, LLC.

Mayberry Hospitality LLC. is a custom, full-service catering company located in Danville, PA. It is fully responsive with static frontend content. It also offers a multi-step form for prospective customers to submit estimate requests. The estimate requests are saved for use by Mayberry in the backend. Once estimate details are finalized, requests can be progressed through a custom scheduler and saved as Google Calendar Events that are shared with the customers so that everything is transparent. All further updates are then re-communicated with the customers for accuracy.
